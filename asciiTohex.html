使用说明

  • ASCII → HEX / 十进制：在左侧输入ASCII文本，右侧自动显示对应的Hex和十进制值
  • HEX → ASCII / 十进制：在中间输入Hex值（每两个字符表示一个字节，空格可选），左右两侧自动转换
  • Hex输入必须是有效的十六进制字符（0-9, A-F, a-f），且长度为偶数
  • 示例：ASCII Hello → HEX 48 65 6C 6C 6F → 十进制 72 101 108 108 111
